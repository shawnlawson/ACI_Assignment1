This is the first assignment for Art, Code and Interactivity, creating an interactive audio and visual piece.

Addons to install for OpenFrameworks:
	- ofxFatLines https://github.com/roymacdonald/ofxFatLines
	- ofxMaxim https://github.com/micknoise/Maximilian

To use:
	- Press and drag the mouse around the window to create lines that make the background visible and the background noise louder
	- Press '[' to decrease the line width and ']' to increase the line width
	- Press 'b' to change the background
	- Press ' ' to clear the window of all the lines
